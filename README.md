<br>

## **Installation**

- Clone this repository

```bash
$ git clone https://github.com/luthfidiqi/Fipay.git
```

- Open Project Folder

```
$ cd Fipay
```

- Instal Depedencies

```
$ npm install
```

- Add env in next.config.js

```
URL_BACKEND=
```

- Run the Project

```
$ npm run dev
```
